Experiment 3 - Alternate Worlds

Description

Description: A valley with a river running through it and clouds overhead. This is mainly a demonstration of autotiling within the context of terrain boundaries (riverbank) and groups of similar tiles (tree clusters). Project can be reseeded as necessary and allows users to edit the generated terrain using the codes below.

Codes:

Underscore ( _ ): Grass or blank

w: Water

t: Tree

Life: Shadows of clouds moving overhead in response to time.

Micro Tileset - Overworld & Dungeon by thkaspar
